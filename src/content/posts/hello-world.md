# 欢迎来到小鱼干的博客 🐟

你好！这是**小鱼干**博客的第一篇文章。

## 关于这个博客

这个博客使用以下技术栈构建：

- **Astro** - 静态站点生成器，性能极佳
- **Vue 3** - 交互式 UI 组件
- **Tailwind CSS** - 原子化样式框架
- **GitHub Pages** - 免费托管

## 主要功能

- 🌓 暗色/亮色主题切换
- 🌐 中英文国际化
- 🔍 全文搜索（`Ctrl + K`）
- 🏷️ 标签分类系统
- 📱 响应式设计
